# 🤖 Amazing ChatBot

An interactive, AI-powered command-line chatbot built as a final project for Stanford University's **Code in Place 2026** program. 

This project demonstrates core programming fundamentals—such as continuous while loops, conditional logic, user input handling, and string formatting—integrated with an AI text-generation backend.

---

## 🚀 Features

*   **Continuous Loop:** Runs seamlessly until the user explicitly types `exit` to quit.
*   **AI Integration:** Leverages the Code in Place `ai` module to dynamically fetch and process smart responses via GPT.
*   **Polished CLI UI:** Features a custom text-art welcome banner and visual separators for a clean user experience.
*   **Custom Prompt Engineering:** Formats user queries into context-specific instructions to ensure friendly, consistent AI responses.

---

## 🛠️ Built With

*   **Python 3**
*   Stanford Code in Place `ai` library (`call_gpt`)
